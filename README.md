## 项目启动说明

1. 需要使用ubuntu系统
2. 请先确保正确安装了Cmake，Opencv，glog等CMakeLists中引用的库
3. 在该目录的终端下输入mkdir build && cd build  && cmake ..
7. 编译完成后 ./demo-1.0
